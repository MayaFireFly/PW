# PW
(Test) The application is for Parrot Wings (PW, "internal money") transfer between system users.

Installation
<br/>
<code>git clone https://github.com/MayaFireFly/PW.git</code>
<br/>
<code>npm install</code>

Start development
<br/>
<code>npm start</code>

Production
<br/>
<code>npm build</code>
